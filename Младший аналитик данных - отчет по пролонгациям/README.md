# Младший аналитик данных: отчет по пролонгациям

Тестовое задание на расчет коэффициентов пролонгации договоров за 2023 год.

## Состав

- `task/` - текст задания.
- `data/` - исходные данные.
- `results/` - Jupyter Notebook с расчетами и Excel-отчет.

## Результаты

- `results/prolongation_analysis.ipynb` - воспроизводимый расчет.
- `results/prolongation_report.xlsx` - итоговый отчет.
